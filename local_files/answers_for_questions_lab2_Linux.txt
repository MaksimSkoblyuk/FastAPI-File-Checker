1. Какой командой можно создать пустой файл?

touch

2. При помощи какой команды можно узнать содержимое каталога?

ls

3. Что такое полный путь и относительный путь?

Полный путь - относительно корня, относительный - относитель текущей директории.

4. Что такое домашний каталог?

Это личный каталог пользователя в ОС, где находятся его данные, настройки и др.

5. Как определить текущий каталог?

Использовать команда pwd

6. Какой ключ для команды ls необходимо применить для того, чтобы отличить файлы от каталогов? Какой ключ этой команды необходимо применить для вывода полного списка содержимого каталога?

Чтобы отличить файлы от каталогов используется ключ -F, для вывода полного списка содержимого каталога используется ключ -a (выводит скрытые файлы).

7. Что такое родительский каталог?

Каталог, находящийся на уровень выше.

8. Какая команда позволяет создавать каталоги? Какой ключ необходимо применить, чтобы создать путь к создаваемому каталогу?

Команда mkdir позволяет создавать каталоги. Чтобы создать путь к создаваемому каталогу, если он не существует, используется ключ -p.

9. Какая команда позволяет вам изменять текущий каталог (перемещаться между каталогами)?

cd

10. Какая команда позволяет переместить файл из одного каталога в другой?

mv

11. Какая команда позволяет создавать копии файлов?

cp

12. Что такое жесткие ссылки? Какая команда позволяет их создавать? Для чего применяются жесткие ссылки?

Жесткая ссылка (hard link) - структурная составляющая файла, описывающая его элемент каталога, "синоним для существующего файла". Позволяет иметь один и тот же файл в нескольких местах, поскольку уникальный индексный дескриптор (inode) совпадает с файлом, на который ссылается жесткая ссылка, при это создается не копия файла, а дополнительный указатель на существующий файл.
Команда ln позволяет создать жесткую ссылку.

13. Какой ключ команды ls необходимо использовать для вывода содержимого каталога с подробным описанием?

Ключ -l.

14. Что такое символьная ссылка? Для чего она применяется? Как создать символьную ссылку?

Символьная или символическая ссылка (symbolic link) - это специальный файл, который является ссылкой на другой файл или католог (целовой файл/каталог). Не содержит внутри себя копии самого файла или директории, на который/ую указывает. Своего рода ярлык на файл/директорию.
Символьную ссылку можно создать с помощью все той же команды ln, указав ключ -s.

15. Каким команда можно удалять файлы и каталоги? Какие ключи необходимо для этого использовать?

Файлы и каталоги можно удалять с помощью команды rm, но для каталогов необходимо указать ключ -r.

16. Какая команда предназначена для создания новой учетной записи пользователя?

useradd/adduser

17. При помощи какой команды можно изменить домашний каталог пользователя?

usermod с ключом -d, а, чтобы переместить содержимое из предыдущей домашней директории в новую, необходимо использовать ключ -m.

18. Какая команда применяется для смены пароля от учетной записи?

Чтобы задать пароль учетной записи - passwd.
Чтобы изменить пароль от учетной записи: usermod --password, при этом необходимо учитывать, что пароль должен изменяться в зашифрованном виде, поэтому следует добавить команду openssl passwd для шифрования пароля.

19. Как удалить пользователя, используя консольую команду? 

Использовать команду deluser или userdel (можно добавить ключ -r для дополнительного удаления личных файлов пользователя, таких как почта и домашняя директория).
